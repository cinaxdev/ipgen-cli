# ipgen-cli

This simple Python CLI tool generates 10 random but realistic-looking public IP addresses.

You can choose a country (Turkey, UK, or USA), and the program will show fake IPs from real IP blocks of that country. These IPs usually appear as real on IP geolocation websites.

## Features

- Country selection (Turkey, UK, USA)
- Realistic-looking public IPs
- Fully terminal-based
- No external libraries required

## Requirements

- Python 3.x

## How to Use

1. Download or clone this repository
2. Open a terminal in the project folder
3. Run the script:

```bash
python ip_generator.py
